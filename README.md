# ISFA - M2 Actuariat - Projet datascience

## Objet

Projet de datascience M2 Actuariat semestre 1.

## Requirements

 Installer Python 3.8.10 avec pip.

 Puis exécuter
> pip install pipenv

Puis utiliser l'environnement virtuel avec

> pipenv shell
> pip install -r requirements.txt   (si nécessaire)
